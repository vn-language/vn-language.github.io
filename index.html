A small, robust programming language

Vanadium is a lightweight high-level systems language designed for clarity, safety, and reliability.

Vanadium is a strictly typed, high-level systems programming language with strong memory management, strict semantics and C interoperability. Designed as a stricter but more modern and less painful alternative of C/C++, Vanadium provides raw power and performance with explicitness and transparency while also implementing a rich standard library with helpers, macros and abstraction of more complex systems.

Examples:

Hello, World! 
include std::io;

frozen fn main() {
    io::print("Hello, World!");
}
Iterate over an array 
include std::io;

frozen fn main() {
    imut names: str[] = ["Jhon", "Tom", "Angela", "Luca"];
    for name in names {
        io::print("Hello, " + name + "!");
    }
}
Unsafe casting 
frozen fn main() {
    imut long_int: int64 = 0xFFFFF;
    imut to_short: int16 = unsafe { long_int as int16 };
    unsafe {
        imut to_short_float: float16 = long_int as float16;
    };
}
Create a Vector2 structure 
struct Vector2 {
    x: float32,
    y: float32,

    frozen fn Vector2(self: *Vector2, x: float32, y: float32) {
        self.x = x;
        self.y = y;
    }
}

frozen fn main() {
    imut my_vec = new Vector2(0.6, 4.7);
}
